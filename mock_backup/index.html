セミナーシステム モックアップ

プロトタイプ概要

セミナーで使用するワークシートを動的に編集できるシステムのUI/UXプロトタイプです。

左のサイドバーから各画面のモックアップを確認できます。

技術仕様
  • フレームワーク: Bootstrap 5.3
  • アイコン: Font Awesome 6.4
  • レスポンシブ: PC/タブレット/スマホ対応
  • ブラウザ: モダンブラウザ対応
実装概要
整理された基本フロー（5ステップ）
  1. 管理者が会社登録 → サインアップ時に会社情報も登録
  2. ユーザー招待 → 未登録ユーザーに招待メール送信（個人メール対応）
  3. セミナー作成 → 管理者がセミナーを新規作成
  4. 参加者選択 → セミナーごとに参加ユーザーを選択・保存
  5. ワークシート共有 → 手動でワークシートを共有（メール通知付き）
主要機能・特徴
  • 個人メール対応：Gmail、Yahoo!メール等でも利用可能
  • 段階的ワークシート開放：手動制御による柔軟な運用
  • メール通知連動：共有と同時に自動通知送信
  • 参加者選択機能：セミナーごとの詳細な参加者管理
  • 金融ダッシュボード風UI：モノトーンで統一されたデザイン
